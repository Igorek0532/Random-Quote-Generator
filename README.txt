1. КРАТКОЕ ОПИСАНИЕ ПРОГРАММЫ
================================================================================

«Random Quote Generator» — это графическое приложение на Python для генерации 
случайных цитат. Программа позволяет:

- Генерировать случайные цитаты из предустановленного списка
- Добавлять собственные цитаты (текст, автор, тема)
- Просматривать историю всех сгенерированных цитат с указанием времени
- Фильтровать историю по автору или по теме
- Автоматически сохранять историю между сеансами работы

Приложение имеет простой и интуитивно понятный интерфейс, подходит для 
ежедневного получения вдохновляющих высказываний или для использования в 
качестве демонстрационного проекта.

6. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И ИХ РЕШЕНИЕ
================================================================================

Проблема 1: Ошибка "No module named tkinter"
Решение (Linux): sudo apt-get install python3-tk

Проблема 2: Ошибка "No such file or directory"
Решение: Проверьте правильность пути к файлу. Убедитесь, что файл существует и
        находится в указанной директории.

Проблема 3: Ошибка кодировки (UnicodeDecodeError)
Решение: Убедитесь, что файл сохранён в кодировке UTF-8. В блокноте Windows
        выберите "Сохранить как" → Кодировка "UTF-8".

Проблема 4: Программа не открывается, нет окна
Решение: Убедитесь, что Python установлен корректно. Попробуйте запустить
        простой скрипт: python -c "print('Hello')"

Проблема 5: Ошибка при импорте tkinter в Windows
Решение: При установке Python убедитесь, что отмечена опция "tcl/tk"
        Переустановите Python с официального сайта

7. ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ И РАБОТЫ ПРОГРАММЫ
================================================================================

ПРИМЕР 1: Генерация случайной цитаты

1. Запустите программу (см. команду выше)
2. Откроется графическое окно "Генератор цитат"
3. Нажмите кнопку «Сгенерировать цитату»
4. В главном поле появится случайная цитата, например:

   "Жизнь — это то, что с тобой происходит, пока ты строишь планы."

5. Ниже отобразится информация об авторе и теме:

   — Джон Леннон (Тема: жизнь)

6. Цитата автоматически добавится в список истории с указанием времени
   генерации (формат: ГГГГ-ММ-ДД ЧЧ:ММ:СС)

ПРИМЕР 2: Добавление собственной цитаты

1. Нажмите кнопку «Добавить свою цитату»
2. В появившемся диалоговом окне введите текст цитаты:
   
   [Текст цитаты:] Программирование — это искусство думать, а не писать код.
   
3. Нажмите OK
4. В следующем окне введите имя автора:
   
   [Автор:] Игорь Ошурков
   
5. Нажмите OK
6. В следующем окне введите тему цитаты:
   
   [Тема:] программирование
   
7. Нажмите OK
8. Система покажет сообщение об успешном добавлении: "Цитата добавлена!"
9. Новая цитата становится доступна для генерации

ПРИМЕР 3: Фильтрация истории по автору

1. Сгенерируйте несколько цитат разных авторов (например, 5-10 раз)
2. В выпадающем списке «Фильтр по автору» выберите, например, "Джон Леннон"
3. В списке истории останутся только цитаты выбранного автора
4. Фильтр применяется автоматически при выборе значения
5. Чтобы сбросить фильтр, нажмите кнопку «Сбросить»

ПРИМЕР 4: Фильтрация истории по теме

1. Сгенерируйте цитаты на разные темы (например, жизнь, мотивация, риск)
2. В выпадающем списке «Фильтр по теме» выберите, например, "мотивация"
3. История покажет только цитаты с выбранной темой
4. Можно комбинировать фильтры по автору и теме одновременно
5. Для сброса всех фильтров нажмите «Сбросить»

ПРИМЕР 5: Сохранение и загрузка истории

1. Сгенерируйте 3-4 цитаты
2. Закройте программу крестиком в углу окна
3. На рабочем столе появится файл history.json
4. Откройте его в блокноте — вы увидите все цитаты в формате JSON:

   [
     {
       "text": "Жизнь — это то, что с тобой происходит...",
       "author": "Джон Леннон",
       "theme": "жизнь",
       "time": "2026-05-05 15:30:22"
     },
     {
       "text": "Будь тем изменением, которое хочешь видеть в мире.",
       "author": "Махатма Ганди",
       "theme": "изменения",
       "time": "2026-05-05 15:31:05"
     }
   ]

5. Запустите программу снова — история автоматически загрузится
6. Сгенерируйте новые цитаты — они добавятся к существующей истории

ПРИМЕР 6: Проверка корректности ввода

При попытке добавить цитату с пустыми полями:

1. Нажмите «Добавить свою цитату»
2. Оставьте текст цитаты пустым и нажмите OK
3. Появится сообщение об ошибке: "Текст цитаты не может быть пустым!"
4. Аналогичная проверка работает для автора и темы
5. Пустые строки не допускаются — программа не позволит их сохранить

9. ОСОБЕННОСТИ РАБОТЫ
================================================================================

- Файл истории: history.json сохраняется на рабочем столе пользователя
- Формат даты: ГГГГ-ММ-ДД ЧЧ:ММ:СС (локальное время компьютера)
- Максимальная длина отображаемой цитаты в истории: 50 символов
  (длинные цитаты обрезаются с добавлением многоточия "...")
- Новые цитаты в истории отображаются сверху
- Фильтры работают одновременно (по автору И по теме)
- При добавлении новой цитаты фильтры автоматически обновляются
- Все цитаты сохраняются в оперативной памяти во время работы программы
- История сохраняется на диск только при генерации новой цитаты

================================================================================
10. СТРУКТУРА ФАЙЛА HISTORY.JSON
================================================================================

Файл history.json имеет следующую структуру:

{
  "quotes": [                    // Массив цитат
    {
      "text": "Текст цитаты",    // Строка, текст цитаты
      "author": "Имя автора",    // Строка, имя автора
      "theme": "Тема цитаты",    // Строка, тема/категория
      "time": "2026-05-05 15:30:22"  // Строка, дата и время генерации
    }
  ]
}

Примечание: Файл создаётся автоматически при первом запуске программы.
Ручное редактирование файла возможно, но не рекомендуется.

12. ВОЗМОЖНОСТИ ДЛЯ РАЗРАБОТЧИКОВ
================================================================================

Программа написана на Python с использованием объектно-ориентированного подхода.
Основные классы и методы:

- QuoteGenerator - главный класс приложения
  - __init__() - инициализация данных и интерфейса
  - create_gui() - создание графического интерфейса
  - generate_quote() - генерация случайной цитаты
  - add_quote() - добавление новой цитаты
  - update_history() - обновление списка истории
  - save_history() - сохранение истории в JSON
  - load_history() - загрузка истории из JSON
  - get_authors() - получение уникальных авторов
  - get_themes() - получение уникальных тем
  - reset_filters() - сброс всех фильтров
